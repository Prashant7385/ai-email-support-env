# Email Support OpenEnv

## 🚀 Description
This is a real-world reinforcement learning environment where an AI agent learns to generate professional responses to customer emails.

## 🎯 Task
The agent must respond to customer emails based on difficulty:
- Easy: General inquiries
- Medium: Order issues
- Hard: Angry refund requests

## 🧠 Action Space
Text response generated by the agent

## 👀 Observation Space
Customer email + difficulty level

## ⭐ Reward Function (0.0 → 1.0)
- Politeness → 0.3
- Helpfulness → 0.3
- Context relevance → 0.4

## 🔄 API Endpoints

- POST `/reset` → Get new email
- POST `/step` → Submit response
- GET `/state` → Episode metadata

## 🛠️ Setup

```bash
cd email_env
venv\Scripts\activate
uvicorn server.app:app --reload
```

## 🧪 Testing & Baseline Inference

Run the baseline agent to see reproducible scores:

```bash
python baseline_inference.py --episodes 3
```

This demonstrates the environment with pre-defined responses and shows grading breakdown.

Example output:
- Average Reward: ~0.55 (varies by episode)
- Score range: 0.0 - 1.0
- Metrics: Politeness, Helpfulness, Relevance

## 🐳 Docker Deployment

```bash
docker build -t email-env .
docker run -p 7860:7860 email-env
```
